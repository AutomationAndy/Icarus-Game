Design Intent

This tool is designed to help track, sort, and organize tames. It is not intended to edit stats or create impossible animals.

Use at your own risk. This tool edits the game save file Mounts.json and is not affiliated with RocketWerkz or the Icarus development team.

Breeding Guide

We can reliably surface parent stats, sex, phenotype, bloodline, and orbital order. We cannot predict exact babies yet, but we can make strong pairing decisions by comparing totals, weak stats, and matching phenotype or bloodline goals.

Suggested naming pattern for in-game pairing: F:P7:V0:F10:P10:R10:T10:A10:I10:Alpha

Pairing Suggestions

Suggestions are based on same-species, opposite-sex pairs, shared phenotype or bloodline goals, and complementary stats.
